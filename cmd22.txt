CATÉGORIE MAJEURE DE DIAGNOSTIC n° 22
Brûlures
Cette présentation comporte trois parties: 
- la description des critères d'entrée dans la CMD n° 22 
- la liste des groupes par ordre croissant des numéros 
- les listes d'actes ou de diagnostics relatives à ces groupes, par ordre croissant de numéros. 
* Une brûlure est considérée comme étendue dès qu'elle atteint 30 % de la surface corporelle. Cette information
doit être codée avec les catégories T31 ou T32 de la CIM-10 pour orienter le RSS dans le GHM correspondant. 
* Depuis la version 6 de la classification, le code L51.2 du syndrome de Lyell a été étendu pour que les cas de
décollement important puissent être orientés dans cette CMD. Seul le code L51.20, permettant de repérer les
décollements égaux ou supérieurs à 30 % de la surface corporelle, oriente le RSS dans la CMD 22 ; le code
L51.29, décrivant les autres formes, reste dans la CMD 09.
